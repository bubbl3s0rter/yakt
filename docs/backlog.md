# Backlog

> Prioritized queue of upcoming work. Pull items into `docs/spec.md` when starting a session.

## Priority

| Tag | Meaning | GitHub sync |
|-----|---------|-------------|
| P0  | Not ready / thinking out loud | Never |
| P1  | Do soon | If tagged `#gh` |
| P2  | Do eventually | If tagged `#gh` |
| P3  | Maybe someday | If tagged `#gh` |

Add `#gh` to a P1–P3 item to sync it to a GitHub issue via `/backlog-sync`.
